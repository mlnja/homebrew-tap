# MLnja Tap

## tama

Multi-agent AI framework — build, run, and trace agent pipelines from the command line.

```bash
brew install mlnja/tap/tama
```

Or explicitly tap first:

```bash
brew tap mlnja/tap
brew install tama
```

Or in a `Brewfile`:

```ruby
tap "mlnja/tap"
brew "tama"
```
